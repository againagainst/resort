### Hot to prepare development environment:
```
python3 -m venv .venv
. .venv/bin/activate
pip install -r reqirements.txt
```

### How to start a basic server:
Terminal: 
```
(.venv) $ python ./basic_rest/app.py
```

VSCode: `F5 (Run Task...) -> "Basic REST Server"`
