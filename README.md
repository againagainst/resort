### Hot to prepare the dev environment:
```
python3 -m venv .venv
./.venv/bin/python3 -m pip install -r reqirements.txt
```

### How to start the REST-test server:
Terminal: 
```
(.venv) $ python ./basic_rest/app.py
```

VSCode: `F5 (Run Task...) -> "Basic REST Server"`
